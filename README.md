# Netflix-data-analysis
"I did an exploratory data analysis on Netflix's content dataset — around 8,800 titles. I cleaned missing values in country and director columns, analyzed content trends over time, and found that Netflix's content additions peaked around 2019-2020, likely tied to the COVID streaming surge. I visualized everything in Python using pandas and matplotlib."

"Movies outnumber TV Shows significantly on Netflix"
"The US dominates content production, followed by India — which shows Netflix's focus on regional content"
"Content additions peaked around 2019-2020, possibly due to COVID streaming surge"
"TV-MA is the most common rating, meaning Netflix targets adult audiences primarily"
